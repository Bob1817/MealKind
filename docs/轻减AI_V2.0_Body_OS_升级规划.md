# 轻减AI V2.0 Body OS 升级规划

## 产品定位
AI Body OS（AI身体管理系统）

### 服务对象
1. 普通减脂用户（习惯驱动）
2. 进阶训练用户（数据驱动）

## 核心升级

### State Engine（状态引擎）
支持：
- 正常训练
- 停训
- 恢复训练
- 受伤
- 出差
- 旅行
- 聚餐
- 节假日

根据状态动态调整：
- 热量目标
- 营养目标
- 今日策略
- AI建议

### Cycle Engine（周期引擎）
支持：
- 减脂期
- 增肌期
- 维持期
- 恢复期

支持训练模板：
- 练三休一
- Push Pull Legs
- Upper Lower
- 自定义模板

### Nutrition Engine（营养引擎）
管理：
- 热量
- 蛋白质
- 碳水
- 脂肪
- 膳食纤维
- 饮水

### Recovery Engine（恢复引擎）
管理：
- 睡眠
- 饮水
- 补剂
- 疲劳评分
- 恢复指数

### Supplement System（补剂系统）
支持：
- 肌酸
- 鱼油
- D3
- 镁
- 电解质
- 自定义补剂

### Strategy Engine（策略引擎）
动态生成：
- 今日饮食策略
- 今日训练策略
- 今日恢复策略
- 今日补剂策略

## 高级模式

普通用户：
- 今日任务
- AI饮食建议
- 体重趋势

进阶用户：
- 宏量营养
- 周期管理
- 恢复指数
- 训练记录
- 补剂管理
